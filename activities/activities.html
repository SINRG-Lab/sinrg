Live Demos

RobotTwin: Human-Robot Collaboration in XR

This demo showcases a mixed reality-based digital twin system for human-robot collaboration in industrial environments. The XR interface enables intuitive visualization of robot tasks, enhancing safety and coordination in factory settings.

XRFab: XR Cleanroom Training Simulator

XRFab simulates cleanroom procedures for semiconductor manufacturing. Trainees interact with wafers and tools using hand tracking on Meta Quest 3, building procedural confidence in a safe, immersive environment.

Interactive XR Assistant for Interventional Radiologists

This demo features a context-aware assistant designed to support interventional radiologists using XR overlays. The system helps visualize anatomical landmarks, procedural steps, and real-time data in a minimally invasive clinical setting.
